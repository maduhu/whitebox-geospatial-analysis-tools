Extract values at XY-coordinates

This tool can be used to extract grid values at specified XY-coordinates. The grid values will be stored in a text file. Additionaly, the min, max and mean values can be computed for the block of 3x3 cells around each of the XY-coordinates.

The user must specify:
The name of the input text file containing the XY-coordinates. The file should contain three (optionally four) columns with successively the id, the X-coordinate, the Y-coordinate and (optionally) a name.
The names of one or more input raster files. For each of the grids, values of the coordinates that are specified in the input text file will be extracted and stored in the output file.
The name of the output text file.
In case the Interpolate 4 cells checkbox is selected, the interpolated value of the four neighrest cells will be computed and stored instead of simply storing the value of one grid cell.
In case the Compute min/max/mean of surrounding cells checkbox is selected, the minimum, maximum and mean values will be computed for the block of 3x3 cells around each XY-coordinate.
