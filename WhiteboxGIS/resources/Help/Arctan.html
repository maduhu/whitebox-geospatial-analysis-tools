Arctan

This tool creates a new raster in which each grid cell is equal to the inverse tangent of the corresponding grid cell in an input raster. Grid cells with No Data values in the input raster will be assigned No Data values in the output raster. The output raster will contain angular data measured in degrees. If desired, you can convert degrees to radians by multiplying the raster by Pi / 180.

This tool can be used with very large raster grids.
