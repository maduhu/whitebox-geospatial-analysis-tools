Olympic filter

This tool performs an olympic filter on a raster image. Olympic filters are closely related mean filters, and therefore, can be used to emphasize the longer-range variability in an image, effectively acting to smooth the image. This can be useful for reducing the noise in an image. The algorithm operates by calculating the average value in a moving window centred on each grid cell. In estimating the average, the highest and lowest values in the neighbourhood are not used. Because averages are highly affected by extremely high and low values, the olympic filter is far less sensitive to shot noise in an image than the mean filter. The median filter is another attractive smoothing alternative, though considerably less computationally efficient.

Neighbourhood size, or filter size, is determined by the user-defined x and y dimensions. These dimensions should be odd, positive integer values, e.g. 3, 5, 7, 9... The user may also define the neighbourhood shape as either squaredor rounded. A rounded neighbourhood approximates an ellipse; a rounded neighbourhood with equal x and y dimensions approximates a circle.

NoDatavalues in the input image are ignored during filtering. When the neighbourhood around a grid cell extends beyond the edge of the grid, NoDatavalues are assigned to these sites. The output raster is of the float data type and continuous data scale.
