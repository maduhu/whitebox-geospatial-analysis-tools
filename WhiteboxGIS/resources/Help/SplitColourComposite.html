Split colour composite

This tool can be used to split an RGB (or aRGB) colour-composite image into three seperate bands of multi-spectral imagery. The user must specify the input image and whether or not to include an output image for the transparency channel (the 'a' channel). The tool creates three or four output images, depending on whether the transparency image is output. Each of the output images will have the same name as the input image with the band appended to it. For example, the input image 'Image' will result in outputs of 'Image_Red', 'Image_Green', 'Image_Blue', and 'Image_A'. Notice that the input image must have a data scale of 'rgb'.

NoData values in the input image are assigned NoData values in the output images. The output rasters are of the float data type and continuous data scale.
