Assign row or column number to grid cells

This tool can be used to create a new raster with the same co-ordinates and demensions (i.e. rows and columns) as an existing base image. Grid cells in the new raster will be assigned either the row or column number or the x- or y-coordinate, depending on the selected option. The user must also specify the name of the base image and the data type (either float or integer). x- or y-coordinate images must be float since most co-ordinate systems require larger numbers than are able to fit within the allowable range of an integer value.

Notice that the functionality of this tool is replicated by the Raster Calculator operations ColumnNumber, RowNumber, XCoordinate, and YCoordinate.
