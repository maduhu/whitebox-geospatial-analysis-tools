Image standard deviation

This tool can be used to calculate the standard deviation value in a raster image. If multiple raster images are input, the algorithm will calculate and report the standard deviation for each input.

See Also:

  • Image Average
  • Image Total

Scripting:

The following is an example of a Python script using this tool:

wd = pluginHost.getWorkingDirectory()
# You may have multiple input files but they must
# be separated by semicolons in the string.
inputFiles = wd + "input1.dep" + ";" + wd + "input2.dep" + ";" + wd + "input3.dep"
args = [inputFiles]
pluginHost.runPlugin("ImageStandardDeviation", args, False)
