Stream network analysis tools

The tool available in Whitebox GAT for performing common stream network analysis operations include the following:

  • Extract streams (automated stream mapping from a DEM)
  • Find main channel stem
  • Find parallel flow directions
  • Horton-Strahler stream order
  • Remove short streams
  • Stream link classification
  • Stream link identifier
  • Stream link length
  • Stream link slope
  • Shreve stream magnitude
